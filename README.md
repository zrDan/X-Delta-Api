## Setup

To run this project, install it locally using npm in backend and frontend

```
$ npm install
$ npm start
```

## End Points

Source: [api](https://d-sneaker-api.herokuapp.com/api/v1) <BR>

```
GET Sneaker List
/sneaker

GET Sneaker By ID
/sneaker/par/:Id
```
